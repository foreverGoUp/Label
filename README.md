 ---------通过扫描二维码打开app中相关文件或网页的二维码内容命名规则-----------
1、若是想打开设备或柜的svg关系图，则文件名规则为：
device + 设备ID 如：device12
cable + 柜ID 如：cable20
注意：ID中不能包含字符“.”
2、若是想打开正常文档文件，则必须以常见文件格式作为文件名后缀。
如:a.txt, b.pdf, c.ppt等等。
